Compétences Développeur Web Junior

HTML

Capacité à créer et structurer des pages web avec HTML.

CSS

Connaissance des styles CSS pour la mise en forme et la présentation des pages web.

JavaScript

Maîtrise des concepts de base de JavaScript pour ajouter de l'interactivité aux pages web.

Bootstrap

Utilisation du framework Bootstrap pour la conception d'interfaces web responsives.

Responsive Design

Capacité à concevoir des interfaces web adaptées à différents appareils et tailles d'écran.

Git

Utilisation de Git pour la gestion de versions et la collaboration sur des projets.

SASS/SCSS

Utilisation de préprocesseurs CSS comme SASS/SCSS pour simplifier et organiser les styles.

Vue.js / React

Connaissance des frameworks JavaScript Vue.js ou React pour la création d'interfaces web dynamiques.

Conception UI/UX

Compréhension des principes de base de la conception d'interfaces utilisateur (UI) et de l'expérience utilisateur (UX).

SEO

Connaissance des techniques de base de SEO pour optimiser le référencement des sites web dans les moteurs de recherche.
